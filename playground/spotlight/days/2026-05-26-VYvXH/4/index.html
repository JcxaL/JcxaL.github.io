Mouse-Reactive Iridescent Button

A pointermove listener writes cursor offset into --cx/--cy CSS custom properties. The ::after pseudo-element — a blurred wide-gamut conic-gradient — translates to that position, making a holographic light source follow your cursor.

HOVER EACH BUTTON TO REVEAL THE HOLO EFFECT
Cursor: —
--cx: —
--cy: —
Glow opacity: 0 (hover to activate)
btn.addEventListener('pointermove', e => {
  const r = btn.getBoundingClientRect();
  btn.style.setProperty('--cx', e.clientX - r.left + 'px');
  btn.style.setProperty('--cy', e.clientY - r.top + 'px');
});

.btn::after {
  background: conic-gradient(in oklch, oklch(72% 0.3 0), ...);
  filter: blur(22px) saturate(1.4);
  mix-blend-mode: screen;
  left: calc(var(--cx) - 80px);  /* tracks cursor */
}
