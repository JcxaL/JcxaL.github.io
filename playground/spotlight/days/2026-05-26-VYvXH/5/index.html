BUTTON VARIANTS

With corner-shape (Chrome 139+)

.card {
  border-radius: 24px;
  corner-shape: squircle;
  /* box-shadow still works! */
}

.btn:hover {
  corner-shape: bevel; /* animate! */
}
transition: corner-shape 0.4s;

Fallback (clip-path squircle)

@supports not (corner-shape: squircle) {
  .card {
    /* SVG superellipse path */
    clip-path: path("M 24,0 ...");
    /* but box-shadow breaks */
  }
}
