FLIP Grid Morph

Three lines of JS capture positions, swap CSS Grid class, then animate from old positions back to new — the GSAP Flip pattern, reimplemented with the Web Animations API. No layout math required.

// 1. Capture  const state = cards.map(c => c.getBoundingClientRect());
// 2. Change  grid.className = newLayout;
// 3. Invert + Play  cards.forEach((c,i) => { const r2 = c.getBoundingClientRect(); const dx = state[i].left - r2.left; c.animate([{transform:`translate(${dx}px,${dy}px) scale(${sw},${sh})`},{transform:'none'}], {duration:600, easing:'cubic-bezier(.4,0,.2,1)'}); })
