The technique: <feTurbulence> generates a procedural noise texture; <feDisplacementMap> reads the red/green channels of that noise and shifts each pixel's X/Y position by scale pixels. The result is text that looks like it's rippling through liquid. Animating baseFrequency via JavaScript produces the flowing illusion — no images, no canvas.
