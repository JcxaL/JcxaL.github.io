Smooth Appear & Disappear

CSS @starting-style + transition-behavior: allow-discrete — filter without JS animation

The trick: @starting-style defines CSS values applied the first time an element transitions from display: none to visible. Without it, the browser has no "from" state for the enter animation — elements would just snap in. transition-behavior: allow-discrete is required to transition display itself, ensuring items stay visible throughout the exit animation before collapsing. No JS animation libraries — just three modern CSS declarations.
