How it works

Each panel below is its own @container. Flipping its switch only sets a --mode custom property on that container — the cards inside react via @container style(--mode: …), with zero JavaScript re-render and zero leakage to sibling panels.

Container Style Queries

A new kind of container query: instead of asking "how wide is my container?", @container style() asks "what custom property value does my container currently have?" — letting a theme cascade down to descendants without touching a single class on them.
