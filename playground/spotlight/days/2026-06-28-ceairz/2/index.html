CSS :has() — Horizontal Accordion

Zero JavaScript. Hover a card to expand it and compress the others using :has() sibling targeting + grid-template-columns transition.

:has(.card:nth-child(N):hover) { grid-template-columns: 1fr 1fr 3fr 1fr 1fr }
:has(.card:hover) .card:not(:hover) { filter: brightness(0.5) }
The parent reacts to descendants — no JS, no classes toggled manually.
