animation-composition: add

Stack multiple @keyframes that each animate transform — they all add together instead of one clobbering the others.

float + spin + scale

replace (default)

Each animation's transform replaces the previous one. Only the last declared animation "wins" — the others are silently cancelled.

add

Each animation's transform is appended to the list. All three run simultaneously and combine — float AND spin AND scale, all at once.
