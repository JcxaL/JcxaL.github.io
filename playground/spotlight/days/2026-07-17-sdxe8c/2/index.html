Anchor Fallback-Aware Panels

CSS anchor positioning can already flip a popover when it would overflow — the new part is knowing which fallback landed, via Chrome 143's container-type: anchored + @container anchored(fallback: …), so the panel's internal layout can react too. This demo layers a JS-measured equivalent so the reflow works in every engine today.

The four corner buttons are pinned to the actual viewport edges — click one and its popover has nowhere to go but flip.
