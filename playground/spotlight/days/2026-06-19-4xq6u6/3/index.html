initial-letter

Real CSS drop caps — 3 variations, no float hacks

Raised Drop Cap

Elegant typography has always been the hallmark of great editorial design. The initial-letter property finally gives CSS the power to create true drop caps — properly aligned, perfectly sized, and entirely without JavaScript or float hacks that plagued web design for two decades.

The property takes a single integer: how many text lines tall the initial letter should be. The browser handles the font metrics, cap height, and baseline alignment automatically.

Sunken Initial

Photography changed everything about how we see the world. The initial-letter: 4 3 syntax enables sunken caps — the letter is 4 lines tall but only sinks 3 lines into the paragraph, leaving one line above the text. This two-value syntax is a 2024 addition now fully supported across browsers.

Combined with border-right and padding, the result is a classic broadsheet newspaper aesthetic without a single line of JavaScript.

Animated Reveal

Because ::first-letter is a pseudo-element, it can be animated with @keyframes. This cap enters with a spring scale and rotation, drawing the eye immediately to the start of the text. The animation fires once on page load — clean, purposeful, memorable.
