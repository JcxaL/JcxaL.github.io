Native Custom Element Tab Bar

A real <spot-tabs> Custom Element — no framework, no shadow DOM, full keyboard support.

01 · Basic horizontal
A clean, accessible tab bar backed entirely by the Custom Elements API — customElements.define(), no build step.
The sliding indicator is one absolutely-positioned div, moved with getBoundingClientRect() math and a CSS transition.
Arrow keys, Home/End, and click all work — role="tablist"/role="tab" wired up in connectedCallback().
02 · Vertical orientation
Same component, same JS — orientation is just a data attribute the element reads on connect.
The indicator swaps from animating translateX+width to translateY+height automatically.
Resize the window — the indicator re-measures and snaps to the active tab instantly.
