Customizable Select

Real, accessible <select> elements — fully keyboard navigable, form-submittable — themed with appearance: base-select. No custom <div> dropdown reimplementation required.

Country picker

Each <option> holds real HTML (a flag + label). A <selectedcontent> inside the closed button auto-mirrors whichever option is selected — closed and open states share one source of markup.

Color swatch picker

Same technique, different content per option: a colored dot swatch instead of a flag. The ::checkmark pseudo-element is repositioned to the trailing edge with plain CSS order.

Assignee picker

An avatar-row select — the kind of control that used to force a hand-rolled combobox. Try it with a keyboard: Tab to focus, arrow keys to move, Enter to pick — all native.
