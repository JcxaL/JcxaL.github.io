Anchor-Positioned Popover Menu

A nested dropdown menu built entirely from the native popover attribute and CSS anchor positioning — no JS positioning math, and focus-trapping / light-dismiss / Escape-to-close all come free from the browser.

Each trigger button carries a CSS anchor-name; each popover sets position-anchor to that name and position-area to place itself relative to it — the "More ▸" submenu anchors to its own trigger and opens to the right, all with the same two properties.

Inspired by css-irl.info — "Anchor Positioning and the Popover API for a JS-Free Site Menu" · UI Spotlight Day 60 · Drop #5
