Highlight-API Word Paint

The CSS Custom Highlight API lets JavaScript mark arbitrary text Ranges and style them purely through ::highlight() — no <span> ever touches the DOM. Three ways to use it: hover, scrub-to-reveal, and live search.

1 · Hover to paint a word

Move your cursor across this sentence and watch each word light up individually as a native text highlight, computed entirely from Range geometry with no wrapper elements involved at all.

2 · Scrub to reveal reading progress

Dragging the slider below marks words as read from the start of this paragraph up to wherever you stop, exactly like a karaoke lyric line or an audiobook progress indicator would.

3 · Live search match

Type a word below to see every matching occurrence highlighted at once across this paragraph, the same mechanism modern in-browser code editors use for fast search-match rendering.
