UI Spotlight · Day 60

Scroll Scopes, Docks, Highlights, Goo & Anchors

Five fresh browser-native patterns turned into live demos: timeline-scope syncing scroll across unrelated DOM branches, a pure-CSS macOS dock via :has(), the CSS Custom Highlight API for span-free word painting, an SVG goo-filter cursor trail, and a nested menu built from the Popover API plus CSS anchor positioning.
