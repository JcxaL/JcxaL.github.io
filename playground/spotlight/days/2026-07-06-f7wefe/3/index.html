DatatypeType a bracket pattern, get a live inline chart.

Edit the paragraph below. Patterns like {b:1,3,7,9}, {l:2,5,3,8,9,1} and {p:82} render live as inline chart glyphs, sized to sit right in the text.

Real Datatype substitutes these patterns for pre-drawn glyphs inside the font's own ligature table (liga/calt), at the text-shaping layer — no JS. This page can't ship a custom variable font binary, so it reimplements the same type-a-pattern-get-a-chart experience with JS-driven inline glyph substitution instead.
