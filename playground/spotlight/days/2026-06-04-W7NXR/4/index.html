Context-Aware
SVG Cursor

The cursor morphs shape based on what it's hovering — circle on default, crosshair on interactive elements, I-beam on text, expand on drag zones, and a spinner on busy elements.

⬤
Default State
The resting cursor: a simple circle dot. Clean, minimal, out of the way.
cursor: default
👆
Interactive Element
Arrow morphs to a filled circle with a ring — universally recognizable as "clickable."
cursor: pointer

Text / Input
Smoothly morphs into a classic I-beam cursor when hovering selectable text or inputs.

Drag Zone
Four-directional arrows expand outward on drag areas — clear affordance for drag interaction.

Precision Target
Classic crosshair morphs for canvas or drawing areas — precision targeting feel.

Loading State
Cursor transforms to a spinning ring to indicate async activity.
cursor: wait
