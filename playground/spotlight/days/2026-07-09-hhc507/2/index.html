Shared-Element Grid-to-Detail Morph

Tap any tile: the native View Transitions API morphs it directly into the full detail card — same element, same paint, no manual FLIP math. Each tile carries a unique view-transition-name; the browser interpolates position, size and aspect-ratio between the two DOM states automatically.

Falls back to an instant swap in browsers without document.startViewTransition.
