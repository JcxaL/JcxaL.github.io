Accessible Swipe-to-Reveal Actions

Drag any row left (mouse or touch) to reveal archive/delete actions, Mail-app style. The same actions are always reachable via the ⋮ menu — the gesture is a shortcut, never the only path, so keyboard and non-pointer users aren't locked out.

Try dragging with a mouse too — this uses Pointer Events, not touch-only listeners.
