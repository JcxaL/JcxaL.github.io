Timeline

The evolution of scroll animation

2020
IntersectionObserver era
Developers used JS observers + CSS classes to trigger animations. Reliable but boilerplate-heavy.
2022
Scroll-Driven Animations spec draft
The CSS Working Group published the scroll animation spec. A single API to rule them all.
2023
Chrome 115 ships it
First major implementation: scroll() and view() timeline functions land in stable Chrome.
2026
Universal browser support
All major browsers ship the spec. Scroll-triggered animations arrive in Chrome 145. The JS scroll library is dead.

Deep Dive

Left & right alternating

view() timeline

Each element creates its own scroll timeline relative to its position in the scroll container's viewport. The animation runs from 0% (element enters) to 100% (element leaves).

📜
animation-range

By setting entry 0% entry 30%, the animation completes by the time 30% of the element is visible. This creates instant, punchy reveals that feel reactive.
