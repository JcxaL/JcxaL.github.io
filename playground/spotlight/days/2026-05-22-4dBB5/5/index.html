Native Details Accordion

Pure CSS accordion: <details name="faq"> + interpolate-size for height: auto — zero JavaScript

What is interpolate-size? +

A new CSS property (Chrome 129+) that allows transitions between a fixed length and an intrinsic size keyword like auto, max-content, or min-content.

Previously, animating height: 0 → height: auto was impossible in CSS — you needed JavaScript hacks. Now it's a single property on :root.

Chrome 129+
How does the name attribute work? +

When multiple <details> elements share the same name attribute, the browser creates a native exclusive group — opening one automatically closes the others.

This shipped in all major browsers in late 2025, eliminating the need for JS-driven accordion state management entirely.

Native HTML
What is ::details-content? +

A new CSS pseudo-element (Chrome 131+) that targets only the non-summary content inside a <details>. This makes it possible to animate the content wrapper independently without wrapping it in an extra <div>.

Combined with interpolate-size, you can write height: 0 → height: auto transitions directly on this pseudo-element.

Chrome 131+
Does it need a fallback? +

Yes for older browsers. The accordion still functions perfectly without the animation — content appears instantly. Progressive enhancement at its best.

For Firefox / Safari fallback, the @supports query can detect interpolate-size support and substitute a JavaScript height calculation if needed.

Progressive
How is the +/× icon animated? +

The .icon span uses a CSS transition: transform .35s cubic-bezier(0.34,1.56,0.64,1) with a spring easing. When details[open] is true, it rotates 45° — turning "+" into "×" with a satisfying bounce overshoot.

CSS Easing
Any JS at all on this page? +

Zero. Not a single line of JavaScript. The open/close state, the mutual exclusion, the height animation, and the icon rotation are all handled by CSS + native HTML semantics.

The total CSS for the accordion technique itself is under 30 lines — the rest is visual polish.

0 JS
Browser support note: interpolate-size requires Chrome 129+ / Edge 129+. The ::details-content pseudo requires Chrome 131+. The name group attribute works in Chrome 120+, Firefox 130+, Safari 17.2+. In unsupported browsers this accordion still works — items open/close instantly without the smooth height animation.
