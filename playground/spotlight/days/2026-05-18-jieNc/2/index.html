container-type: inline-size

Opt a parent into being a named or anonymous container that children can query against.

@container (max-width: 520px)

Respond to the container's width, not the viewport. Works inside grid, flex, or any layout.

container-name: card

Name containers so nested @container rules can target a specific ancestor, not just the nearest one.

@container style()

Style queries (Chrome 111+) let you query a container's CSS custom property values too.
