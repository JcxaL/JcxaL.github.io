shape-outside

CSS text wrapping that follows the shape, not the box.

shape-outside: circle(50%)

Circle Float

◉ circle(50%)

When you apply shape-outside: circle(50%) to a floated element, the text wraps along the circular edge rather than the rectangular bounding box. Pair it with border-radius: 50% and you get text hugging a perfect disc.

This is one of the oldest shape-outside values and the most reliable. The shape is defined in the float's coordinate system and the browser calculates line-box intersections at each text line.

It works in all modern browsers including Firefox, Chrome, Safari and Edge. The shape can also receive a margin via shape-margin to add breathing room between the shape edge and the nearest text character.

Try resizing this window — the text reflows naturally while still respecting the curved exclusion zone.

shape-outside: polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%)

Polygon Float — Diamond

◆ polygon

The polygon() function accepts any number of vertices, letting you define a custom exclusion shape. The text wraps around the actual diamond boundary — not the square bounding box that surrounds it.

You can pair shape-outside with clip-path using the same polygon coordinates so the visual shape and the text-exclusion shape match exactly. That's exactly what this demo does.

Polygons are powerful for editorial layouts: pull-quote triangles, starburst callouts, angled hero sections where text needs to fill a wedge shape precisely.

Complex polygons with many vertices are still performant — the browser recomputes intersections only when the layout changes, not on every scroll frame.

shape-outside: ellipse(120px 80px at 50% 50%)

Ellipse Float

⬭ ellipse

ellipse(rx ry at cx cy) defines an elliptical exclusion. Here the float is 240×160px — a landscape rectangle — but text wraps as though a horizontal ellipse floats in it.

The ellipse radii (120px, 80px) match exactly half the float's dimensions, so the exclusion fills the entire element. Change either radius to add straight margins on two sides.

Ellipses feel more organic than rectangles. Magazines have used asymmetric photo placements for centuries; shape-outside: ellipse() lets you bring that to the web without image masks or complex hacks.

shape-outside: inset(0 0 0 0 round 40px)

Inset Rounded Rectangle

▣ inset + round

The inset() function creates a rectangle with optional rounding. The round keyword takes the same syntax as border-radius, so this shape has 40px-rounded corners — and the text follows those curves exactly.

This is the shape-outside equivalent of a card component. Float a rounded card to the right and let prose wrap around the rounded corners for a polished editorial feel.

You can also inset the shape inside the float's box — inset(10px 10px 10px 10px) adds 10px of "clearance" on all sides before the text gets near the element, without needing shape-margin.

Combined with shape-margin you have fine-grained control over the gap between the visual edge and the closest text glyph.
