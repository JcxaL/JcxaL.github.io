SVG Filter Technique

Each word uses feTurbulence + feDisplacementMap.

The turbulence generates a noise field; the displacement map uses it to offset pixel coordinates. Animating scale from 0 → peak → 0 creates the warp effect.

baseFrequency controls warp size. type: turbulence = smooth; fractalNoise = organic.

Hover to distort

Hover or tap each word · Scale and frequency vary per filter
