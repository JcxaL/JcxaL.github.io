@starting-style Toast Notifications

Enter and exit animations with pure CSS transitions — no classList toggling, no animation tricks.

/* 1. Define the FROM state on first insertion */
@starting-style {
  .toast { opacity: 0; transform: translateX(100%) }
}

/* 2. Define the TO state normally */
 .toast { opacity: 1; transform: none; transition: opacity .3s, transform .3s }

/* 3. Exit: use transition:display allow-discrete (or hidden attr) */
 .toast[hidden] { display: flex!important; opacity: 0; transform: translateX(100%) }
