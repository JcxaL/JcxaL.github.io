CSS Spotlight Hover

A cursor-following radial gradient punches a "spotlight" through a dark overlay — 2 lines of JS total

🔦

The Spotlight Effect

A fixed dark overlay with a radial gradient transparent hole tracks your cursor. The hole reveals content underneath as if lit by a flashlight.

⚡

Only 2 Lines of JS

The mousemove listener only calls style.setProperty('--cx', e.clientX + 'px') and the same for Y. All the visual work is done in CSS.

🎯

GPU Composited

Because background with custom properties is compositor-friendly and pointer-events: none is set, there's no layout or paint thrashing on move.

🍭

Soft vs. Hard Edge

The gradient stop ratio controls softness. transparent 0%, black calc(radius * 1.3) creates a wide feathered bloom; a 1:1 ratio gives a sharp flashlight edge.

🔮

:has() Integration

Add body:has(button:hover) .overlay { opacity: 0 } to remove the effect when users hover interactive elements — no extra event listeners needed.

📐

Invert: Torch Mode

Flip the gradient — black 0%, transparent at radius — for a "torch beam" that illuminates the cursor area while leaving the rest of the page dark.

/* CSS — the entire effect */ .overlay { position: fixed; inset: 0; pointer-events: none; background: radial-gradient( circle var(--radius) at var(--cx, 50%) var(--cy, 50%), transparent 0%, rgba(0,0,0,0.97) calc(var(--radius) * 1.3) ); } /* JS — 2 lines */ document.addEventListener('mousemove', e => { el.style.setProperty('--cx', e.clientX + 'px'); el.style.setProperty('--cy', e.clientY + 'px'); });
