Highlight API Text Painting

Text gets colored ranges without touching the DOM — Range objects registered through CSS.highlights and styled with ::highlight(), the browser's native "type to highlight" mechanism.

1 · Type to highlight

Every year the web platform ships a few features that quietly replace a native JavaScript library. The Custom Highlight API is one of them: it paints matches directly onto existing text nodes, so a live search box can highlight every native occurrence of a query without wrapping a single character in a span.
