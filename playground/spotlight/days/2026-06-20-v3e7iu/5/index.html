Native Dialog Animations

5 <dialog> open/close animations with @starting-style — no JS animation

@starting-style · slide-up

Slide Up Modal

This native <dialog> element slides in from below using CSS @starting-style — the browser transition fires on entry without any JS animation code.

 @starting-style · scale-in

Scale In Modal

Scaling from 80% with a spring cubic-bezier gives a satisfying "pop in" effect. The spring bounce (1.3 overshoot) is from a single CSS parameter.

 @starting-style · slide-left

Slide Left Modal

Directional slides communicate context — this dialog feels like it's arriving from a side panel or drawer, reinforcing spatial orientation.

 @starting-style · 3D flip

3D Flip Modal

A subtle 3D rotateX gives depth on entry. Combined with a spring easing, it feels like the dialog is physically unfolding from above.

 @starting-style · blur-in

Blur-In Modal

Transitioning filter:blur() alongside scale creates a focus-pulling effect — the scene sharpens as the dialog arrives. Pure CSS, no GSAP.
