Zero-JS Focusgroup Toolbar

The new focusgroup HTML attribute turns any group of buttons into a single tab stop with native arrow-key roving focus — the "roving tabindex" pattern, built into the browser. A matching roving-tabindex script keeps it working everywhere today.
