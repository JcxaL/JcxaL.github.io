Shape() Morph Deck

Three live shapes built entirely with the CSS shape() function — curved, responsive clip-paths that keyframe-animate via registered custom properties. No SVG, no polygon hacks.

If your browser doesn't yet support clip-path: shape(), the boxes above render as plain rounded rectangles — everything else on the page still works.
