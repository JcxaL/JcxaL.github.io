CSS Subgrid: Align Across Column Boundaries

With grid-template-rows: subgrid, a child grid can inherit the row tracks of its parent — meaning elements in different columns align their internal content to the same horizontal baselines. This eliminates the classic "each card defines its own height" problem that made CSS grids feel rigid.
