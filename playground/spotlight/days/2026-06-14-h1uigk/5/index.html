Liquid Gooey Tabs

Overview

The liquid indicator uses an SVG feGaussianBlur + feColorMatrix filter to create a gooey merge effect. When the pill slides, edges appear to melt into each other before snapping.

Features

Built entirely in CSS + vanilla JS — no libraries. The goo effect comes from blurring then thresholding alpha via feColorMatrix values="… 22 -10". The spring transition is pure cubic-bezier(.34,1.56,.64,1).

Reviews

"The indicator feels alive — it oozes between tabs instead of sliding rigidly. That single filter line changes everything." The overshoot in the cubic-bezier makes the pill bounce past, then settle.

Pricing

Free to use — it's just CSS. Three separate tab bar styles are shown here: pill, icon-grid, and underline bubble, all using the same goo technique with different accent gradients.

SVG feGaussianBlur + feColorMatrix thresholds alpha so overlapping shapes merge into liquid blobs. A spring cubic-bezier makes the indicator overshoot and settle. Three styles, one trick.
