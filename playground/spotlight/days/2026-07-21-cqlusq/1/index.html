Scroll Equalizer

Three rows of CSS-only bars, each tied to page scroll position via animation-timeline: scroll(root). Negative animation-delay per bar phase-shifts the same keyframe timeline, turning one wave into a chasing equalizer. No JavaScript drives the motion — only the scrollbar does.
