Eight Distinct Reveal Effects

Every element uses a different @keyframes animation, all driven by the same scroll mechanism. No duplicated JS — just CSS that listens to the viewport.

Effect 1

Fade Up

The classic: opacity 0→1 + translateY 60px→0. Elegant, universally readable.

from: opacity:0 translateY(60px)
to: opacity:1 translateY(0)
Effect 2

Scale Pop

Grows in from 65% scale. Perfect for cards and media that need to "arrive".

from: scale(0.65) opacity:0
to: scale(1) opacity:1
Effect 3

Slide from Left

Sweeps in 80px from the left. Great for lists and feature rows.

from: translateX(-80px)
animation-range: entry 10% 50%
Effect 4

Slide from Right

Mirror of effect 3 for alternating layouts.

from: translateX(80px)
animation-range: entry 10% 50%
Effect 5

Rotate Up

A subtle -8° rotation + scale(0.85) creates a playful, energetic entrance.

from: rotate(-8deg) scale(.85)
to: rotate(0) scale(1)
Effect 6

3D Flip In

Perspective X-rotation gives a satisfying flip from a reclined angle.

from: perspective(600px) rotateX(-40deg)
to: rotateX(0)
Scroll Reveals Everything

How It Works

The browser creates a ViewTimeline tracking when this element's bounding box enters and exits the scrollport. The animation's progress percentage is driven by that timeline instead of elapsed time.

.elem {
  animation: fade-up linear both;
  animation-timeline: view();
  animation-range: entry 0% entry 50%;
}
📜
Scroll position
= animation progress
view()  scroll()
Two timeline types

view() tracks this element's journey through the viewport — animation-range controls which part of that journey maps to 0%–100%. scroll() tracks the container's total scroll from top to bottom.
