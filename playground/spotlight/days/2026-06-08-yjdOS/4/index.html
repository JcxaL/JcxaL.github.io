The technique: On mousemove, compute cursor's offset from the card's centre normalised to −1→+1. Multiply by maxTilt to get rotateX / rotateY. Apply via inline style.transform. The glare is a radial-gradient whose centre CSS variable tracks the same normalised position. On mouseleave, re-add a CSS transition and reset the transform — giving the spring-back feel.
