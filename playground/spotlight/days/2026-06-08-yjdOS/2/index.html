CSS Subgrid Alignment

Grid cards that align across arbitrary content heights — zero JavaScript

grid-template-rows: subgrid
⚠️ Your browser doesn't support CSS Subgrid yet. Chrome 117+ / Firefox 71+ / Safari 16+ required.
Row alignment: synchronized

Each card has content of different lengths. Without subgrid, cards stack independently — prices and buttons float at different heights. With grid-template-rows: subgrid, the parent grid's row tracks are inherited by each card's children, forcing them to align across the entire row.

Without subgrid: price floats wherever content ends
With subgrid: price always on same row across all cards
/* Parent grid — defines shared row tracks */
.grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  gap: 1.5rem;
}

/* Each card spans 6 parent row tracks and uses subgrid */
.card {
  display: grid;
  grid-row: span 6;
  grid-template-rows: subgrid; ← the magic
  align-items: start;
}
