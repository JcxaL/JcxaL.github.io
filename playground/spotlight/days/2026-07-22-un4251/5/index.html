Hover-to-Play Media Cards

Static thumbnails that lazy-load into looping motion only once they're near the viewport — desktop hovers to play, touch taps to play, and scrolling away pauses everything automatically. (Canvas-generated loops stand in for video, since this playground ships zero external assets.)

Inspired by Cult UI — Hover Video Player component (cult-ui.com) · UI Spotlight Day 66 · Drop #5
