CSS linear() Easing

The linear() timing function (CSS 2024) lets you encode any easing curve as comma-separated stops — spring, bounce, elastic, anything — in native CSS.

Spring linear(0, .006, .025, .101, ...)
Bounce linear(0, .004, ..., 1.03, ..., 1)
Ease In-Out ease-in-out (classic)
Elastic Snap linear(0, -.014, -.024, ..., 1)
Linear linear (no easing)

Technique

Spring overshoot
Bounce decelerate
Classic ease-in-out
Elastic oscillation
No easing (linear)
