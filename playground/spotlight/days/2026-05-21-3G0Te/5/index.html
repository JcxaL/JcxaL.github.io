Native Popover Panel
This panel uses popover="manual" — it won't close on outside click, only via the explicit close button or pressing Esc. The backdrop is a pure CSS ::backdrop pseudo-element.

The enter/exit animations use @starting-style to define the "before" state, enabling CSS-only transition from hidden → visible.
How it works: Add popover to any element and popovertarget="id" to a button — the browser handles accessibility, top-layer z-index, focus trap, and light-dismiss. Combine with @starting-style for entrance animations that work even when the element goes from display:none to visible.
