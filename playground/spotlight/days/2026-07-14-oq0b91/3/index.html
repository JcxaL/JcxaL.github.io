Contour-Hugging Type

The border and the paragraph's text-wrap trace the exact same hand-drawn path — sampled once, applied twice: as a live border-shape stroke where supported, and as a shape-outside polygon for the float everywhere else.

The border of this card and the paragraph that wraps beside it share one identical contour. Instead of a generic circle() or blob wrap, the shape is sampled point-by-point from a single path function, then fed to two different CSS features at once: border-shape reshapes the actual box border where the browser supports it, while shape-outside — broadly supported today — carves the matching exclusion zone so text reflows around the identical silhouette. Change the contour style in the panel and watch both the stroke and the paragraph reflow together, live, with no layout thrash.
