Intrinsic Typography

No media queries. Each text element's size, weight and line-height is stored as a @keyframes curve from 0% (empty container) to 100% (full width) — a ResizeObserver writes the live container width into --tt-bind, and a paused animation is scrubbed to that exact frame via a negative animation-delay. This paragraph's font-size and line-height interpolate continuously with the box width — one CSS rule, zero breakpoints, driven purely by the element's own intrinsic size.
