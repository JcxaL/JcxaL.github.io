Paint Worklet Reactive UI

These backgrounds aren't images or CSS gradients — they're drawn live by a CSS Houdini Paint Worklet, a JS function running off the main thread that paints straight into background: paint(name). Move your pointer over each card; the grain drifts and the aurora blob chases you.

Your browser doesn't support the CSS Painting API — showing a static gradient fallback instead. Try latest Chrome or Edge.
