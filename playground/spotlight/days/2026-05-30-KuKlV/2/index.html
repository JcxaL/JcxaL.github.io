01 — Design

Cross-DOM
Scroll Sync

CSS timeline-scope lets a scroll timeline defined in one branch of the DOM drive animations in a completely separate branch — without a single line of JavaScript.

02 — Motion

Named View
Timelines

Each section declares a view-timeline-name. The common ancestor expands the scope so sibling subtrees can reference those timelines as animation-timeline values.

03 — Layout

Sticky Panel
Driven by Scroll

The right panel stays fixed while you scroll the left column. Each card fades in and out exactly as its corresponding section enters and exits the viewport — pure CSS.

04 — Color

Entry & Exit
Range Keywords

The animation-range: entry 0% exit 100% spans the full time a section is visible — from first pixel entering to last pixel leaving the viewport.

05 — Future

No JS
Required

This entire interaction — the linked sidebar, the smooth crossfades, the synchronized state — runs entirely in CSS. The browser's compositor does all the work.
