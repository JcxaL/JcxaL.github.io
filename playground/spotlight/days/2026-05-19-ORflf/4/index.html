Drop #4 · Micro-interaction

SVG Gooey
Filter Magic

The gooey effect uses an SVG filter combo: feGaussianBlur softens edges, then feColorMatrix thresholds the alpha channel — turning blurred overlaps into liquid merges.

Demo 1 — Merging blobs
Hover adjacent buttons to see them merge like liquid. The SVG filter applies to the container — all children share the same goo field.
Demo 2 — Liquid pill navigation
The active indicator is a div that slides under each item. The goo filter makes the transition look like liquid mercury sliding across the bar.

Demo 3 — Burst on click
Click to spawn a blob that expands and dissolves. The gooey filter merges the spawned blob with the button momentarily, then releases it.
Demo 4 — Magnetic hover
Buttons gently follow the cursor within their bounding box, creating a tactile pull effect without the goo filter.
Demo 5 — Gooey dot cluster
Hover any dot to expand it — adjacent dots stay connected via the goo filter, making them feel like one living organism.
