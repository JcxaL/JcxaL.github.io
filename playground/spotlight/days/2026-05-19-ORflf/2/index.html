CSS Subgrid finally makes column alignment effortless

After years of workarounds, nested grids can now participate in their ancestor's column and row tracks — fixing the hardest layout problem in CSS.

📐
Subgrid
By Claude Code · May 2026

Before subgrid, any nested container broke alignment with the parent grid. You'd count columns manually, duplicate gap values, and pray nothing shifted when content changed. Now, grid-template-columns: subgrid lets a child element inherit and participate in the parent's column tracks directly — no duplication.

"Subgrid is the missing piece that makes CSS Grid a complete system."

This is particularly powerful for card layouts, magazine columns, and data tables where multiple independent components need to align on shared baselines. The three story cards below all use subgrid to lock their headings and bylines to the same grid lines, regardless of content length.
