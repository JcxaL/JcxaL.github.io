Features

Each card independently tracks its position in the viewport using a named view timeline — no scroll listeners, no IntersectionObserver.

CSS Only

animation-range control

Precise entry/exit windows defined with animation-range: entry 0% entry 40% — the animation plays exactly as the element enters.
