# JccL - Personal Website

This is the personal website of JccL, built with [Next.js](https://nextjs.org) and deployed on GitHub Pages.

## About

This website serves as my personal space on the web where I share my projects, thoughts, and experiences in software development and technology.

## Getting Started

To run this project locally:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Modern, responsive design built with Next.js and Tailwind CSS
- Personal portfolio showcase
- Blog section for sharing thoughts and experiences
- Project gallery
- Contact information

## Tech Stack

- [Next.js](https://nextjs.org) - React framework for production
- [Tailwind CSS](https://tailwindcss.com) - Utility-first CSS framework
- [TypeScript](https://www.typescriptlang.org) - Type-safe JavaScript
- Deployed on [GitHub Pages](https://pages.github.com)
